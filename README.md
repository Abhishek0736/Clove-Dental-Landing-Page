# Clove Dental Landing Page

## Project Overview
A fully responsive landing page for Clove Dental built as part of the Web Development Intern Assessment. The project demonstrates proficiency in HTML, CSS, and Vanilla JavaScript by translating Figma designs into a functional, interactive website.

## Live Demo
https://clove-dental-landing-page-one.vercel.app/(#) 

## Technologies Used
- **HTML5** - Semantic markup structure
- **CSS3** - Responsive styling and animations
- **Vanilla JavaScript** - Interactive functionality and form validation

## Sections Implemented (6 Total)
1. **Navigation Bar** - Sticky header with Book Appointment and Call buttons
2. **Hero Section** - Consultation form with image and call-to-action
3. **Stats Strip** - Animated marquee showcasing clinic statistics
4. **Root Canal Treatment Cards** - Service offerings with pricing
5. **Patient Testimonials** - Real transformation showcase with infinite scroll
6. **Appointment Booking Form** - Bottom section with form validation
7. **Footer** - Company information and links

## Key Features
- ✓ Fully responsive design (Mobile: 320px - Desktop: 1200px+)
- ✓ Interactive form validation with captcha verification
- ✓ Real-time visual notifications for user actions
- ✓ Smooth scroll navigation
- ✓ Animated statistics marquee
- ✓ Hover effects and transitions
- ✓ Cross-browser compatible

- Mobile: 320px - 480px

## Form Validation Features
- Name field validation
- 10-digit phone number validation
- 4-digit captcha verification
- Terms & conditions checkbox
- Real-time error messages
- Success notifications

## Interactive Elements
- Navbar scroll effects
- Book Appointment button (scrolls to form)
- Call button (initiates phone call)
- Stats items (clickable with notifications)
- RCT treatment cards (hover effects + form navigation)
- Patient testimonial cards (click for details)
- Footer links (smooth scroll to top)

## Installation & Setup
1. Clone the repository
```bash
git clone <repository-url>
```

2. Navigate to project directory
```bash
cd Assisment
```

3. Open `index.html` in your browser or use a local server
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
